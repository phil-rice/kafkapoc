# Transformation Module

## Overview
This module handles data transformation operations, specifically focused on XML transformations.

## Components

### XmlTransformExecutorForMapStringObject
- **Purpose**: Executes XML transformations on Map<String, Object> data structures
- **Location**: `src/main/java/com/hcltech/rmg/execution/validation/xml/XmlTransformExecutorForMapStringObject.java`
- **Usage**: Transforms XML data into Map structures for easier processing

## Key Features
- XML-based data transformation
- Support for Map<String, Object> data structures
- Integration with validation and execution pipelines

## Dependencies
Refer to `pom.xml` for Maven dependencies required by this module.

## Integration
This module works in conjunction with other execution modules to transform incoming XML messages into processable data formats.
